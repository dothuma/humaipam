## Critical rules for next agent

### Never skip the chain
```
.r → .rf → .fnf → src
```
Code is always last. This session violated this for jsonwf. Fix it first.

### Always use fqnp
Every artifact starts with:
```
% fqnp: path/to/artifact
```
Every source function is wrapped:
```python
# fqnp: subapp/src/file.py:function_name
def function_name(...):
    ...
# fqnp: subapp/src/file.py:function_name:end
```

### Always deliver as .huma patch
Never give mv commands for files that do not exist on disk.
Always: generate .huma → humadese.py applies → files land on disk.

### Session context
Read .session on start. If context lost — ask Creator:
```bash
source ~/_humaipam_v5/.session
```

### hu:/ma: notation
Creator uses hu: for natural language.
Agent responds in ma: predicate, executes, returns ma: result.
Mistral translates ma: back to hu: when needed.

### Predicate prompts
Full vocabulary in: md/PREDICATE_PROMPTS_0405_2030.md
Two-agent loop: md/ARTICLE_TWO_AGENT_LOOP_0405_2015.md

### Tokens
Use predicate frames. Natural language is expensive and drifts.
One predicate = 6-8x fewer tokens than equivalent natural language.
